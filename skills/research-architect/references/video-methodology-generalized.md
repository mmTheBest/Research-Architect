# Generalized Video-Derived Methodology

The original video-derived method contained a useful workflow but an unsafe surface framing. The safe abstraction is:

1. Learn from concrete examples rather than starting from vague aspiration.
2. Decompose examples into contribution logic, work package, evidence sequence, and manuscript structure.
3. Reconstruct the core logic before proposing changes.
4. Design a bounded contribution that can be supported by evidence.
5. Build the paper from a claim-evidence chain rather than from decorative writing.
6. Use figure and paragraph structure as a communication scaffold, not as copyable content.

This upgraded skill keeps those ideas but expands them into a full research lifecycle:

```text
raw idea -> research candidates -> literature map -> confirmed spine -> study design -> evidence bank -> citation bank -> first draft -> audit
```

The main change is that exemplar learning is no longer the center of the skill. It is one component inside a broader research-construction workflow.
