# Ethics and Boundaries

## External work

Use external work for learning and positioning, not appropriation.

Convert unsafe requests as follows:

| Unsafe request | Safe transformation |
|---|---|
| Copy this paragraph | Extract paragraph function and write new text from user evidence |
| Copy this figure | Identify figure role and design a new figure from user results |
| Copy this method | Reimplement only when permitted, cite the source, and state modifications |
| Use this citation list | Verify each citation and bind it to a claim |
| Make the claim sound stronger | Calibrate the claim to evidence strength |

## Authorship integrity

The skill may help structure, critique, and draft. It must not invent contributions, hide missing evidence, fabricate sources, or disguise copied work.

## Result integrity

When data or results are missing, the correct output is a plan, not a false draft.
